# Interactive Marketing Analytics App (Streamlit)

Author: Halima Ladan  

## Overview
This project is an interactive Streamlit application designed to explore marketing performance data and support data-driven decision-making.

The app allows users to visualize trends, analyze campaign performance, and quickly identify patterns in time series data without requiring technical expertise.

---

## Key Features

- Interactive time series visualization of marketing data  
- Exploration of trends and performance over time  
- Simple interface designed for non-technical stakeholders  
- Fast and intuitive data exploration for decision-making  

---

## Use Case

This application bridges the gap between data analysis and business decision-making by enabling stakeholders to:

- Monitor campaign performance  
- Identify trends and anomalies  
- Support data-driven marketing decisions  

---

## Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib / Plotly (if applicable)
